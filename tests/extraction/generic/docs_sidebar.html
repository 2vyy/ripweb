API Reference

The following functions are part of the public ripweb API surface. They are stable unless marked experimental.

fetch(url: &str) -> Result<Bytes>

Fetches a URL using the configured HTTP client with connection pooling and domain politeness.

let bytes = ripweb::fetch("https://example.com").await?;

extract(bytes: &[u8]) -> Result<String>

Extracts structured Markdown from raw HTML bytes. Applies the boilerplate nuke list, candidate scoring, and family-aware rendering.

let markdown = ripweb::extract(&bytes)?;
println!("{}", markdown);

Parameters

  • bytes — raw HTML input, must be valid UTF-8 or will be transcoded
  • content_type — optional HTTP Content-Type header for charset detection

Returns

A Result<String> where the String is a stable Markdown representation of the page content.
